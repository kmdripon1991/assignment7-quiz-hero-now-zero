What is event loop in JavaScript? How it works?

JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks.
The event queue pushes the tasks from the task queue to the call stack. At some point during the event loop, the runtime starts handling the messages on the queue, starting with the oldest one. Each message is processed completely before any other message is processed.
